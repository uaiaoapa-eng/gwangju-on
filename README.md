# 광주ON

> 시민이 묻고, 행정이 답하고, 광주가 함께 결정해요.
> 전남광주통합특별시 시민을 위한 **AI 통합 행정 · 시민참여 플랫폼** — 기획 시연용 프로토타입.

**[📋 상품 기획서 (PRODUCT.md)](PRODUCT.md)**

## 데모에서 볼 수 있는 것

- 🤖 **AI 민원 도우미** — 민원·맛집·관광까지 광주에 관한 질문에 답해요. 기본은 30개 주제 지식베이스 데모, **Claude API 키를 연동하면 모든 질문에 진짜 AI가 답해요** (챗 화면 '연동 설정')
- 💡 **시민 제안 · 공감** — 제안 등록, 공감 100 도달 시 부서 답변 플로우
- 🗳️ **정책 라이브 투표** — 주민참여예산 우선순위 실시간 투표
- 📊 **행정 투명 대시보드** — 민원 처리율·예산 집행 공개 UI
- 🏘️ **우리동네 광주** — 5개 자치구 이슈 보드

## 디자인

모바일 우선 리디자인 — 하단 탭바(홈·AI민원·제안·투표·동네), 데스크톱(≥1080px)에선 왼쪽 네비게이션 레일.
**LUXROBO/MODI 디자인 시스템** 기반 (Pretendard 내장, 브랜드 블루 `#2F6BFF`).

## 실행

빌드 도구·의존성 없음 — `index.html`을 브라우저로 열면 끝. 폰트가 파일에 내장돼 **오프라인에서도** 동작해요.

## 구조

```
index.html   # 단일 파일 앱 (HTML + CSS + 바닐라 JS + 폰트 내장)
PRODUCT.md   # 상품 기획서
```

챗·제안/공감·투표·연동 설정 상태는 모두 브라우저 `localStorage`에 저장돼요.
Claude API 키는 **이 저장소에 절대 커밋하지 마세요** — 사이트 화면의 '연동 설정'으로 각 기기 브라우저에만 저장하는 방식이에요.

## 고지

본 사이트의 수치·기관 연계는 **시연용 가상 데이터**이며, 전남광주통합특별시(광주시)의 공식 서비스가 아니에요.
