# 광주ON 🌿

> 시민이 묻고, 행정이 답하고, 광주가 함께 결정합니다.
> 광주광역시 공무원·시민을 위한 **AI 통합 행정 · 시민참여 플랫폼** — 기획 시연용 프로토타입.

**[📋 상품 기획서 (PRODUCT.md)](PRODUCT.md)**

## 데모에서 볼 수 있는 것

- 🤖 **AI 민원 도우미** — 폐기물·주정차·청년월세·전입신고 등 민원을 물으면 담당 부서와 절차를 안내 (데모는 키워드 매칭, 정식판은 Claude API)
- 💡 **시민 제안 · 공감** — 제안 등록, 공감 100 도달 시 부서 답변 플로우 시각화
- 🗳️ **정책 라이브 투표** — 주민참여예산 우선순위 실시간 투표
- 📊 **행정 투명 대시보드** — 민원 처리율·예산 집행 공개 UI
- 🏘️ **우리동네 광주** — 5개 자치구 이슈 보드

## 실행

빌드 도구 없음 — `index.html`을 브라우저로 열면 끝.

```bash
# 또는 로컬 서버
python -m http.server 5600
```

## 구조

```
index.html   # 단일 페이지
styles.css   # 디자인 (무등산 그린 + 빛고을 옐로)
script.js    # 챗 데모 · 제안/공감 · 투표 · 카운트업 (localStorage)
PRODUCT.md   # 상품 기획서
```

## 고지

본 사이트의 모든 수치·AI 응답·기관 연계는 **시연용 가상 데이터**이며, 광주광역시의 공식 서비스가 아닙니다.
